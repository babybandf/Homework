📐 013 - 等腰直角三角形中BD=DP的证明

进度:
1/1
📋

题目

在等腰直角三角形 △ABC 中，∠BAC = 90°，AB = AC。直线 MN 过点 A 且 MN // BC。点 D 在直线 MN 上（不与点 A 重合），DB 与 AC 交于点 P，且 ∠BDE = 90°。求证：BD = DP。

1

作辅助线，构造正方形 AFDG

过点 D 作 DF ⊥ AB 于点 F，作 DG ⊥ AC 于点 G。

因为 MN // BC，且 △ABC 是等腰直角三角形（∠BAC = 90°），所以 ∠BAM = ∠ABC = 45°，∠CAN = ∠ACB = 45°。即 MN 方向使得 D 到 AB 和 AC 的距离相等：DF = DG。

又因为 DF ⊥ AB，DG ⊥ AC，且 AB ⊥ AC，所以四边形 AFDG 是矩形。结合 DF = DG 可知 AFDG 是正方形，从而 DF = DG 且 ∠FDG = 90°。

💭 思考问题

? 为什么 D 到 AB 和 AC 的距离相等？
✓ 因为 MN // BC，而 △ABC 是等腰直角三角形（∠A=90°, AB=AC），所以 ∠ABC = ∠ACB = 45°。由平行线性质，MN 与 AB、AC 形成的角也是 45°，因此 D 到两边距离相等。
? 为什么 AFDG 是正方形而不是矩形？
✓ 矩形有三个直角（∠AFD=∠FDG=∠DGA=90°），又因为 DF = DG（已证邻边相等），所以是正方形。正方形的关键性质：四边相等、四角都是直角、对角线互相垂直平分且相等。
2

证明角度相等关系

由已知条件 ∠BDE = 90° 和正方形性质 ∠FDG = 90°，可得：

∠BDF + ∠FDP = ∠PDG + ∠FDP = 90°

因此 ∠BDF = ∠PDG（同角的余角相等）。

又由辅助线作法可知：
- ∠BFD = 90°（DF ⊥ AB）
- ∠PGD = 90°（DG ⊥ AC）
- DF = DG（已证正方形性质）

💭 思考问题

? "同角的余角相等"在这里怎么用？
✓ ∠BDE = 90° 意味着射线 DB 与 DE 垂直；∠FDG = 90° 意味着 DF 与 DG 垂直。∠FDP 是公共部分，从两个 90° 中分别减去它，剩下的 ∠BDF 和 ∠PDG 必然相等。
? ASA 全等的三个条件分别是什么？
✓ 角：∠BFD = ∠PGD = 90°；边：DF = DG（公共）；角：∠BDF = ∠PDG（已证）。注意顺序：角-边-角，夹边必须是两角的公共边。
3

利用 ASA 全等得出结论

在 △BDF 和 △PDG 中：

  ① ∠BFD = ∠PGD = 90°
  ② DF = DG（正方形性质）
  ③ ∠BDF = ∠PDG（已证）

根据 ASA（角边角） 判定定理：

$\triangle BDF \cong \triangle PDG$

由全等三角形对应边相等，得：
BD = DP

💭 思考问题

? 为什么选择 ASA 而不是 SAS 或 SSS？
✓ 我们最容易得到的是两组角相等（直角 + 余角关系）和一条夹边相等（DF=DG）。这正好符合 ASA 的模式。SAS 需要两边及其夹角，SSS 需要三边，都不如 ASA 直接。
? 这个结论对 P 在 AC 延长线上也成立吗？
✓ 成立！辅助线的构造和推理过程不依赖 P 在 AC 边上的具体位置，只要 P 在直线 AC 上即可。无论 P 在 AC 上、CA 延长线上还是 AC 延长线上，同样的方法都可以证明 BD = DP。
4

情形2：P 在 CA 延长线上（配置图）

当点 P 在 CA 的延长线上时，同样过点 D 作 DF ⊥ AB 于点 F，作 DG ⊥ AC 于点 G。

因为 MN // BC，且 △ABC 是等腰直角三角形（∠BAC = 90°），所以 ∠BAM = ∠ABC = 45°，∠CAN = ∠ACB = 45°。点 D 到 AB 和 AC 的距离相等：DF = DG。

又因为 DF ⊥ AB，DG ⊥ AC，且 AB ⊥ AC，所以四边形 AFDG 是矩形。结合 DF = DG 可知 AFDG 是正方形，从而 DF = DG 且 ∠FDG = 90°。

💭 思考问题

? 情形2和情形1的辅助线构造有何不同？
✓ 辅助线构造完全相同！都是过 D 作 DF ⊥ AB 于 F，DG ⊥ AC 于 G。DF = DG 和正方形 AFDG 的结论也与情形1一致，不依赖于 P 的具体位置。
? P 在 CA 延长线上时，DB 与 AC 如何相交？
✓ 此时 P 在 CA 的延长线上（即 A 的左侧），DB 需要延长后才能与 AC 的延长线交于点 P。图中以虚线表示 AC 的延长线，红色线段 DP 表示延长后的交点。
5

情形2：ASA 全等证明

由已知条件 ∠BDE = 90° 和正方形性质 ∠FDG = 90°：

∠BDF + ∠FDP = ∠PDG + ∠FDP = 90°

因此 ∠BDF = ∠PDG（同角的余角相等）。

又 ∠BFD = ∠PGD = 90°，DF = DG，故：

$\\triangle BDF \\cong \\triangle PDG$ (ASA)

由全等三角形对应边相等，得：
BD = DP

💭 思考问题

? 情形2的证明逻辑与情形1有何异同？
✓ 完全相同！正方形性质 → 同角余角相等 → ASA 全等 → BD = DP。核心逻辑链条不依赖 P 的位置，唯一的区别是 P 在 CA 延长线上，但三角形 BDF 和 PDG 的对应关系不变。
? 图中蓝色和绿色分别标记了什么？
✓ 蓝色标记了 △BDF（包含直角符号和角弧），绿色标记了 △PDG（同样包含直角符号和角弧）。这两个三角形是全等的，对应边 BD 和 DP 即为所求。
6

情形3：P 在 AC 延长线上（配置图）

当点 P 在 AC 的延长线上时，同样过点 D 作 DF ⊥ AB 于点 F，作 DG ⊥ AC 于点 G。

同理可证 DF = DG，且 AFDG 为正方形，从而 DF = DG 且 ∠FDG = 90°。

此时 P 在 C 的右侧，DB 与 AC 的延长线交于点 P。图中以虚线表示 AC 延长线，红色线段 DP 表示交点。

💭 思考问题

? 情形3的辅助线构造与前两种情形相同吗？
✓ 完全相同！三种情形都使用相同的辅助线：DF ⊥ AB 于 F，DG ⊥ AC 于 G。这说明辅助线的选择是普适的，不依赖于 P 的位置。
? 三种情形下结论都是 BD = DP 吗？
✓ 是的！无论 P 在线段 AC 上、在 CA 延长线上还是在 AC 延长线上，通过相同的辅助线和证明方法，都能得到 BD = DP。这体现了数学证明的统一性。
7

情形3：ASA 全等证明

由 ∠BDE = 90° 和 ∠FDG = 90°，得 ∠BDF = ∠PDG。

又 ∠BFD = ∠PGD = 90°，DF = DG，故：

$\\triangle BDF \\cong \\triangle PDG$ (ASA)

因此：
BD = DP

💭 思考问题

? 三种情形使用同一种证明方法，这说明了什么？
✓ 说明辅助线 DF ⊥ AB、DG ⊥ AC 的构造和 ASA 全等的证明方法具有普适性。无论 P 在 AC 上的哪个位置（包括延长线），只要 P 在直线 AC 上，都可以通过构造正方形 AFDG 来证明 BD = DP。
? 图中蓝色三角形和绿色三角形的关系？
✓ 蓝色 △BDF 和绿色 △PDG 全等（ASA）。对应边 BD（蓝色标记）和 DP（绿色标记）即为需要证明相等的两条线段。角弧和直角符号标出了全等的关键条件。

解题技巧

  • • "见垂直想正方形"：当一点到两条互相垂直的线段距离相等时，常可构成正方形作为桥梁
  • • "有直角找互余"：多个直角共存时，通过同角的余角相等建立角度联系
  • • "垂线法"：涉及线段相等证明时，过某点向两边作垂线将分散条件集中处理
