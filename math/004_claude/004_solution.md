# 004 - 求∠BMC的度数

## 题目

在△ABC中，∠BAC = ∠BCA = 44°，点M为△ABC内一点，且∠MCA = 30°，∠MAC = 16°，求∠BMC的度数。

## 解题过程

### 第一步：分析已知条件

由题目条件可以计算：
- ∠BAC = ∠BCA = 44° → AB = BC（等腰三角形）
- ∠ABC = 180° - 44° - 44° = **92°**
- ∠BAM = ∠BAC - ∠MAC = 44° - 16° = **28°**
- ∠BCM = ∠BCA - ∠MCA = 44° - 30° = **14°**
- 在△AMC中：∠AMC = 180° - 16° - 30° = **134°**

**目标：** 在△BMC中，∠MCB = 14°，只要确定∠MBC就能求出∠BMC。

### 第二步：构造辅助线

以AC为边，在AC上方（B同侧）作**等边△ACE**，连接BE。

性质：
- AE = CE = AC（等边三角形三边相等）
- ∠EAC = ∠ECA = 60°（等边三角形每个角都是60°）

### 第三步：证明△ABE ≅ △CBE（SSS）

- AB = CB（△ABC是等腰三角形，∠BAC = ∠BCA）
- AE = CE（△ACE是等边三角形）
- BE = BE（公共边）

∴ △ABE ≅ △CBE（SSS）

由全等得：**∠AEB = ∠CEB = ∠AEC ÷ 2 = 60° ÷ 2 = 30°**

### 第四步：证明△ABE ≅ △AMC（ASA），得AB = AM

观察△ABE和△AMC，发现：

① ∠EAB = ∠EAC - ∠BAC = 60° - 44° = **16°** = ∠MAC ✓

② AE = AC（等边三角形，AE = AC）✓

③ ∠AEB = **30°** = ∠MCA ✓

∴ **△ABE ≅ △AMC（ASA）**

由全等得：**AB = AM**

### 第五步：利用等腰△ABM求角

由第四步得 AB = AM，所以△ABM是等腰三角形。

在△ABM中：
- ∠BAM = 28°（第一步已算）
- AB = AM（等腰）

∴ ∠ABM = ∠AMB = (180° - 28°) ÷ 2 = **76°**

所以 ∠MBC = ∠ABC - ∠ABM = 92° - 76° = **16°**

### 第六步：计算∠BMC

∠BMC = 360° - ∠BMA - ∠AMC = 360° - 76° - 134° = **150°**

或等价地：在△BMC中，∠BMC = 180° - ∠MBC - ∠MCB = 180° - 16° - 14° = **150°**

## 最终答案

$$\boxed{\angle BMC = 150°}$$

## 知识点归纳

1. **等腰三角形性质**：两底角相等 → 两腰相等
2. **等边三角形性质**：三边相等，三角都是60°
3. **全等三角形（SSS）**：三边分别相等的两个三角形全等
4. **全等三角形（ASA）**：两角及其夹边分别相等的两个三角形全等
5. **等腰三角形求角**：等腰三角形两底角相等
6. **辅助线构造**：构造等边三角形来制造全等条件
