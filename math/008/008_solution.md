# 008题：手拉手模型——等边三角形的全等证明

## 题目

【加油站】在学习全等三角形的知识时，数学兴趣小组发现这样一个模型：它是由两个共顶点且顶角相等的等腰三角形构成，在相对位置变化时，始终存在一对全等三角形。通过查询资料，他们得知这种模型称为"手拉手模型"，兴趣小组进行了如下操作：

**(1) 观察猜想**：如图①，已知△ABC，△ADE均为等边三角形，点D在边BC上，且不与点B，C重合，连接CE，易证△ABD ≅ △ACE，进而判断出AB与CE的位置关系是 ______。

**(2) 类比探究**：如图②，已知△ABC，△ADE均为等边三角形，连接CE，BD，若∠DEC = 60°，试证明点B、D、E三点共线。

**(3) 解决问题**：如图③，已知点E在等边△ABC的外部，并且与点B位于线段AC的异侧，连接AE，BE，CE。若∠BEC = 60°，AE = 3，CE = 2，请求出BE的长。

---

## 第(1)问解答

### 分析思路

这是典型的"手拉手模型"：两个等边三角形共顶点A，通过旋转可以得到全等三角形。

### 证明 △ABD ≅ △ACE

**已知条件**：
- △ABC是等边三角形 → AB = AC，∠BAC = 60°
- △ADE是等边三角形 → AD = AE，∠DAE = 60°

**关键观察**：

∠BAD = ∠BAC - ∠DAC = 60° - ∠DAC

∠CAE = ∠DAE - ∠DAC = 60° - ∠DAC

**所以**：∠BAD = ∠CAE

**全等判定**（SAS）：
- AB = AC（等边三角形边长相等）
- ∠BAD = ∠CAE（已证）
- AD = AE（等边三角形边长相等）

**结论**：△ABD ≅ △ACE

### 判断AB与CE的位置关系

由全等可知：∠ABD = ∠ACE

因为△ABC是等边三角形，所以∠ABC = ∠ACB = 60°

即 ∠ABD = 60°

所以 ∠ACE = 60°

观察∠BCE = ∠BCA + ∠ACE = 60° + 60° = 120°（当D在BC上时）

或者考虑：∠ABC + ∠BCE = 60° + 120° = 180°？不对，这需要重新分析。

**正确分析**：

由△ABD ≅ △ACE，得∠ABD = ∠ACE = 60°（因为∠ABC = 60°，D在BC上，所以∠ABD就是∠ABC）

在△ABC中，∠ABC = 60°

所以∠ACE = 60° = ∠BAC

这说明CE与AB被AC所截，内错角相等（∠ACE = ∠BAC = 60°）

**答案**：**AB // CE**（AB平行于CE）

## 第(2)问解答

### 分析思路

要证明B、D、E三点共线，只需证明∠BDE = 180°，即证明∠ADB + ∠ADE = 180°，或证明∠BDE为平角。

### 证明过程

**第一步：证明△ABD ≅ △ACE**

与(1)同理：
- AB = AC（等边△ABC）
- AD = AE（等边△ADE）
- ∠BAD = ∠CAE = 60° - ∠DAC

所以△ABD ≅ △ACE（SAS）

**第二步：利用全等性质**

由全等得：∠ADB = ∠AEC

**第三步：利用已知条件∠DEC = 60°**

因为△ADE是等边三角形，所以∠AED = 60°

观察点E处的角：

∠AEC = ∠AED + ∠DEC = 60° + 60° = 120°

所以∠ADB = ∠AEC = 120°

**第四步：证明共线**

在△ADE中，∠ADE = 60°（等边三角形内角）

所以∠BDE = ∠ADB + ∠ADE = 120° + 60° = 180°

**结论**：∠BDE = 180°，所以**B、D、E三点共线**

## 第(3)问解答

### 分析思路

已知条件：
- △ABC是等边三角形
- E在△ABC外部，与B在AC异侧
- ∠BEC = 60°
- AE = 3，CE = 2

要求：BE的长度

**关键思路**：使用截长法！既然猜测 BE = AE + CE，我们可以尝试在线段 BE 上截取一段等于 CE，证明剩下的一段等于 AE。

### 构造辅助线

在线段 BE 上截取一点 F，使得 BF = CE。连接 AF。

### 证明△ABF ≅ △ACE

**已知条件**：
- 因为△ABC是等边三角形，所以 AB = AC，∠BAC = 60°
- 观察四边形 ABEC，由于点 E 与点 B 在 AC 异侧，且 ∠BAC = 60°，∠BEC = 60°
- 设 AC 与 BE 交于点 O。在△ABO 和△CEO 中，∠AOB = ∠COE（对顶角），∠BAO = 60°，∠CEO = 60°（已知）

**关键观察**：
根据三角形内角和，剩下的第三个角相等：∠ABO = ∠ECO。即 ∠ABF = ∠ACE。

**全等判定**（SAS）：
- AB = AC（等边三角形）
- ∠ABF = ∠ACE（已证）
- BF = CE（辅助线作法）

**结论**：△ABF ≅ △ACE

### 分析△AFE

由全等可知对应边相等：**AF = AE = 3**

由全等可知对应角相等：**∠BAF = ∠CAE**

**观察∠FAE**：
∠FAE = ∠FAC + ∠CAE

代入 ∠CAE = ∠BAF：
∠FAE = ∠FAC + ∠BAF = ∠BAC = 60°

所以，△AFE 中：
- AF = AE = 3（等腰三角形）
- ∠FAE = 60°

**结论**：一个角是 60° 的等腰三角形是等边三角形，所以 △AFE 是等边三角形，**EF = AE = 3**

### 计算结果

**BE = BF + EF = CE + AE = 2 + 3 = 5**

所以答案是：**BE = 5**

## 解题技巧总结

### 1. 手拉手模型的核心特征
- 两个共顶点的等腰（或等边）三角形
- 顶角相等
- 通过旋转可得到全等三角形

### 2. 三点共线的证明方法
- 证明夹角为180°
- 证明两线段在同一直线上

### 3. 构造辅助线的策略
- 当题目给出部分条件时，主动构造完整的模型
- 利用前几问的结论指导后续解题
