001 - 不等式整数解问题

进度:
1/5
📋

题目

已知 a > 0，且不等式 1 < ax < 2 恰有三个正整数解，则当不等式 2 < ax < 3 含有最多的整数解时，正数 a 的取值范围为__________。

1

分析第一个不等式

由 a > 0，不等式 1 < ax < 2 可化为 1/a < x < 2/a。设该不等式恰有三个正整数解，记为 n, n+1, n+2。

💡 关键隐含条件

三个解必须是连续的。因为不等式的解集是一个连续区间 (1/a, 2/a)，一个连续区间不可能跳过中间的整数。例如，区间 (2.3, 5.7) 包含的正整数是 3, 4, 5——它们是连续的：

 2.3 │ 3   4   5 │ 5.7
─────┼───────●───┼────
      ↑ 区间起点  ↑ 区间终点

如果有人说解是 3, 5, 7（不连续），那 4 和 6 也必然在区间内，解集就会变成 3, 4, 5, 6, 7，不止三个了。因此恰好三个正整数解，必然是三个连续整数 n, n+1, n+2。

需要满足的条件：

  • n-1 ≤ 1/a < n（保证 n 是最小的正整数解）
  • n+2 < 2/a ≤ n+3（保证 n+2 是最大的正整数解）

💭 思考问题

? 为什么需要 n-1 ≤ 1/a < n 这个条件？
✓ 这个条件确保 n 是最小的正整数解。如果 1/a < n-1，则 n-1 也会是解；如果 1/a ≥ n，则 n 不是解。
? 从条件能得出 a 的什么范围？
由 1/a < n 得 a > 1/n；由 n+2 < 2/a 得 a < 2/(n+2)。所以 1/n < a < 2/(n+2)。
2

确定 n 的值

检验不同的 n 值，找出满足 1/n < a < 2/(n+2) 的 n：

n 1/n 2/(n+2) 有解？ a 的范围
1 1 2/3 ≈ 0.67 ❌ —
2 0.5 0.5 ❌ —
3 1/3 ≈ 0.33 0.4 ✅ 1/3 < a < 2/5
4 0.25 1/3 ≈ 0.33 ✅ 2/7 < a < 1/3

情况 A（n=3）：解为 3,4,5

  • 2 ≤ 1/a < 3 → 1/3 < a ≤ 1/2
  • 5 < 2/a ≤ 6 → 1/3 ≤ a < 0.4
  • 交集：1/3 < a < 2/5

情况 B（n=4）：解为 4,5,6

  • 3 ≤ 1/a < 4 → 1/4 < a ≤ 1/3
  • 6 < 2/a ≤ 7 → 2/7 ≤ a < 1/3
  • 交集：2/7 < a < 1/3

💭 思考问题

? 为什么 n = 1 和 n = 2 没有解？
✓ n=1 时区间 (1, 2/3) 为空；n=2 时区间 (0.5, 0.5) 也为空。需要 1/n < 2/(n+2) 才有解。
? n=3 和 n=4 都满足条件，接下来怎么办？
分别对两种情况分析第二个不等式 2 < ax < 3 的整数解个数，再比较哪个更多。
3

分析第二个不等式（分类讨论）

不等式 2 < ax < 3 可化为 2/a < x < 3/a，区间长度为 1/a。现在对两种情况进行分类讨论。

情况 A（n=3，a ∈ (1/3, 0.4)）：

  • 2/a 的范围是 (5, 6)
  • 3/a 的范围是 (7.5, 9)

整数解分析：

  • 当 a ∈ (1/3, 3/8)：2/a < 6 且 3/a > 8 → 6, 7, 8（3个）
  • 当 a ∈ [3/8, 0.4)：3/a ≤ 8 → 6, 7（2个）

情况 B（n=4，a ∈ (2/7, 1/3)）：

  • 2/a 的范围是 (6, 7)
  • 3/a 的范围是 (9, 10.5)

整数解分析：

  • 当 a ∈ (2/7, 0.3)：2/a < 7 且 3/a > 10 → 7, 8, 9, 10（4个） 🏆 最多
  • 当 a ∈ [0.3, 1/3)：3/a ≤ 10 → 7, 8, 9（3个）

对比总结：

情况 a 的范围 第二不等式最多解 对应子区间
A（n=3） (1/3, 0.4) 3个 (1/3, 3/8)
B（n=4）🏆 (2/7, 1/3) 4个 (2/7, 0.3)

结论：情况 B（n=4）在 a ∈ (2/7, 3/10) 时第二不等式有 4 个整数解，多于情况 A 的 3 个，因此取情况 B 的范围。

💭 思考问题

? 为什么需要分别讨论 n=3 和 n=4 两种情况？
两个 n 值都满足"恰有三个正整数解"的条件，但它们在数轴上对应不同位置的区间，所以第二个不等式会有不同的整数解个数，必须分开分析。
? 情况 B 中 a 的范围为什么是 (2/7, 3/10)？
要得到 4 个整数解 7,8,9,10，需要 2/a < 7（即 a > 2/7）且 10 < 3/a（即 a < 3/10）。再与第一不等式的约束 (2/7, 1/3) 取交集得 (2/7, 3/10)。
🎯

思路梳理

最终答案

2/7 < a < 3/10

此时不等式 2 < ax < 3 有 4 个整数解 7, 8, 9, 10

解题流程

1
将 1 < ax < 2 化为 1/a < x < 2/a，三个解为连续整数 n, n+1, n+2
2
分类讨论：n=3（a∈(1/3,2/5)）和 n=4（a∈(2/7,1/3)）均满足条件
3
分析 2 < ax < 3：情况 A 最多 3 个解，情况 B 最多 4 个解
4
情况 B 胜出：a ∈ (2/7, 3/10) 时第二不等式有 4 个整数解

验证（a = 0.29）：

第一不等式：1/a≈3.45，2/a≈6.90 → 整数解 4,5,6 ✓ | 第二不等式：2/a≈6.90，3/a≈10.34 → 整数解 7,8,9,10（4个）✓

知识点

一元一次不等式 整数解计数 参数范围确定

解题技巧

  • • 转化法：将含参数的不等式转化为关于 x 的区间形式
  • • 端点分析法：通过分析区间端点位置确定整数解个数
  • • 范围交集法：多个条件限制时取各范围的交集
