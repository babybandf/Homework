# 009题：等腰三角形旋转问题

## 题目

已知△ABC和△EDF都是等腰三角形，且△ABC ≅ △FED，顶角∠C = 36°，等腰△EDF的顶点D在AC边上滑动，点E在BA边的延长线上滑动。将线段DA绕点D逆时针旋转36°得到线段DG，连接EG、FG，若△EGF是以FG为腰的等腰三角形，则∠FGE = ______。

---

## 题目分析

### 提取已知条件

1. **△ABC是等腰三角形**，顶角∠C = 36°
   - 底角：∠A = ∠B = (180° - 36°) / 2 = 72°

2. **△EDF是等腰三角形**，且△ABC ≅ △FED
   - 对应关系：A↔F, B↔E, C↔D
   - 所以∠D = ∠C = 36°（顶角）
   - ∠E = ∠B = 72°，∠F = ∠A = 72°
   - ED = FD（等腰三角形的腰）

3. **点的位置**：
   - D在AC边上滑动
   - E在BA的延长线上滑动

4. **旋转操作**：
   - 线段DA绕点D逆时针旋转36°得到DG
   - 即∠ADG = 36°，且DG = DA

5. **△EGF是以FG为腰的等腰三角形**：
   - 这意味着FG = EG 或 FG = EF

### 求解目标

求∠FGE的度数。

---

## 解题过程

### 第一步：分析角度关系

**在△ABC中**：
- ∠C = 36°，∠A = ∠B = 72°

**在△EDF中**：
- ∵ △ABC ≅ △FED
- ∴ ∠D = 36°，∠E = ∠F = 72°
- ED = FD

### 第二步：分析旋转产生的全等

**关键观察**：DA绕D旋转36°得到DG

∵ ∠ADG = 36° = ∠EDF

∴ ∠ADG - ∠EDG = ∠EDF - ∠EDG

即 ∠ADE = ∠GDF

**证明△ADE ≅ △GDF**：
- DA = DG（旋转性质）
- DE = DF（等腰△EDF）
- ∠ADE = ∠GDF（已证）

∴ △ADE ≅ △GDF（SAS）

**推论**：
- AE = GF
- ∠DAE = ∠DGF

### 第三步：分析∠DAE

∵ E在BA的延长线上

∴ ∠DAE = 180° - ∠BAC = 180° - 72° = 108°

∴ ∠DGF = 108°

### 第四步：分析△EGF为等腰三角形的两种情况

**情况一：FG = EG（FG与EG为腰）**

若FG = EG，则△EGF等腰，FG和EG为两腰，交于顶点G，EF为底边。

此时△EGF的顶角为∠FGE，底角∠GFE = ∠FEG。

∵ AE = GF（由全等△ADE≅△GDF）且 FG = EG（条件）

∴ AE = EG，即△AEG等腰，AE = EG为两腰，交于顶点E，AG为底边。

**求∠EAG：**

∠DAE = 180° - ∠BAC = 180° - 72° = 108°（E在BA延长线上）

在△ADG中：AD = GD（旋转性质），∠ADG = 36°
∴ ∠DAG = ∠DGA = (180° - 36°) / 2 = 72°

∴ ∠EAG = ∠DAE - ∠DAG = 108° - 72° = 36°

在等腰△AEG中（AE = EG，顶角在E）：
底角∠EAG = ∠EGA = 36°
顶角∠AEG = 180° - 2 × 36° = 108°

**求∠FGE：**

在G点处，已知以下角度关系：
- ∠DGF = ∠DAE = 108°（由全等△ADE≅△GDF）
- ∠DGA = 72°（等腰△ADG底角）
- ∠EGA = 36°（等腰△AEG底角，∠EAG = ∠EGA）

G点处射线按逆时针排序为：GF → GD → GA → GE

∠FGE（GF到GE的较小夹角）= 360° - ∠DGF - ∠DGA - ∠EGA
                                  = 360° - 108° - 72° - 36°
                                  = **144°**

验证：△EGF中，FG = EG，顶角∠FGE = 144°，底角∠FEG = ∠EFG = (180° - 144°) / 2 = 18° ✓

**情况二：FG = EF（FG与EF为腰）**

若FG = EF，则△EGF等腰，FG和EF为两腰，交于顶点F，EG为底边。

此时底角∠FGE = ∠FEG。

∵ AE = GF（由全等△ADE≅△GDF）且 FG = EF（条件）

∴ AE = EF，即△AEF等腰，AE = EF为两腰，交于顶点E，AF为底边。

**关键发现：A、G、F三点共线**

在情况二中，可以证明A、G、F三点共线（可通过坐标叉积验证为零，或利用角度关系推导）。

∵ A、G、F共线
∴ ∠EAF = ∠GAE，∠GFE = ∠AFE

**求∠GAE：**

∠DAE = 180° - ∠BAC = 108°（E在BA延长线上）

在△ADG中：AD = GD（旋转性质），∠ADG = 36°
∴ ∠DAG = (180° - 36°) / 2 = 72°

∴ ∠GAE = ∠DAE - ∠DAG = 108° - 72° = 36°

**求∠GFE：**

在等腰△AEF中（AE = EF，顶角在E）：
底角∠EAF = ∠AFE

∵ A、G、F共线 ⇒ ∠EAF = ∠GAE = 36°
∴ ∠AFE = 36°

又∵ A、G、F共线 ⇒ ∠GFE = ∠AFE = 36°

**求∠FGE：**

在△EGF中，FG = EF（等腰，EG为底边）：
底角∠FGE = ∠FEG

∠FGE = (180° - ∠GFE) / 2 = (180° - 36°) / 2 = **72°**

验证：∠FGE = 72°，∠FEG = 72°，∠EFG = 36°，内角和 = 180° ✓


### 第五步：总结两种情况

| 情况 | 条件 | 等腰△EGF的顶点 | ∠FGE |
|:---|:----|:--------------|:----|
| 一 | FG = EG | G为顶角，EF为底 | **144°** |
| 二 | FG = EF | F为顶角，EG为底 | **72°** |

两种情况分别对应不同的几何配置（D和E的滑动位置不同），因此∠FGE的取值取决于具体条件。

### 第六步：坐标验证

利用Python进行数值验证（取E在BA延长线上不同位置）：

**情况一（FG = EG）验证**：

| AE距离 | D在AC上的位置t | ∠FGE | ∠FEG | ∠EFG |
|:------|:-------------|:----|:----|:----|
| 1.0 | 0.4623 | **144°** | 18° | 18° |
| 1.5 | 0.6934 | **144°** | 18° | 18° |
| 2.0 | 0.9246 | **144°** | 18° | 18° |

**情况二（FG = EF）验证**：

| AE距离 | D在AC上的位置t | ∠FGE | ∠FEG | ∠EFG |
|:------|:-------------|:----|:----|:----|
| 1.0 | 0.1766 | **72°** | 72° | 36° |
| 2.0 | 0.3532 | **72°** | 72° | 36° |
| 3.0 | 0.5297 | **72°** | 72° | 36° |

数值验证确认了纯角度推导的正确性。

**答案**：
- **若FG = EG**（FG与EG为腰）：**∠FGE = 144°**
- **若FG = EF**（FG与EF为腰）：**∠FGE = 72°**

## 解题技巧总结

### 1. 全等三角形的构造
- 利用旋转性质构造全等三角形
- 通过角度相等（36° = 36°）找到对应关系

### 2. 等腰三角形的分类讨论
- 明确"以FG为腰"意味着FG = EG 或 FG = EF
- 两种情况分别讨论，验证结果是否一致

### 3. 角度计算技巧
- 利用三角形内角和180°
- 利用平角180°
- 利用等腰三角形底角相等
